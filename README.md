##Norm

Norm[{a, b, c} - {x, y, z}]


##Euclidean Distance
EuclideanDistance[{a, b, c}, {x, y, z}]
Formula= dist((x, y), (a, b)) = √(x - a)² + (y - b)²

##Squared Euclidean Distance
EuclideanDistance[{a, b, c}, {x, y, z}]^2
